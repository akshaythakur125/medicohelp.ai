# Educational Strategy

## Core Philosophy
MedicoHelp.ai delivers free, high-quality MBBS revision through Telegram using a static content library of 190 topics across 19 subjects. The system is designed to operate with zero recurring costs while maintaining pedagogically sound content delivery.

## Educational Modes
| Mode | Subjects | Target Audience |
|------|----------|----------------|
| Comprehensive | All 19 | All students |
| First Year MBBS | Anatomy, Physiology, Biochemistry | First-year students |
| Final Year Revision | Medicine, Surgery, OBG, Pediatrics | Final-year students |
| NEET PG Revision | 12 core clinical subjects | PG aspirants |
| INICET High-Yield | 12 core clinical subjects | INICET aspirants |
| Emergency 5-Min | All (rapid formats only) | Quick revision |

## Content Formats (13)
- MCQ (6-7 line clinical vignettes)
- Image-Based Questions
- Concise Notes
- Clinical Cases
- Rapid Revision
- Practical/Viva
- Exam News
- Residency Tips
- PYQ Concepts
- Flashcards
- True/False
- One-Liner Recall
- Mnemonics

## Pedagogical Enhancements
1. **Memory Anchors** — Subject-specific mnemonics, visual associations, and memory tricks appended to explanations
2. **Clinical Pearls** — High-yield pearls from ward-round and exam contexts
3. **Concept Cross-References** — Links between related topics across subjects (e.g., Pharmacology → Biochemistry for CYP450)
4. **Bloom's Taxonomy Tagging** — Content tagged by cognitive level (recall, understand, apply, analyze)
5. **Distractor Analysis** — Each MCQ option gets a realistic wrong-option explanation with clinical reasoning

## Spaced Repetition
- Default cooldown: 7 days between repeats
- Weak topics (accuracy <60%): 3-day cooldown for faster resurfacing
- Strong topics (accuracy ≥80%): 7-day cooldown
- Three-layer dedup: 100-entry recent set, 500-title strategy set, performance-based cooldown

## Engagement Features
- Streak tracking with milestone messages (1, 7, 30, 100 days)
- Daily challenge MCQ with accuracy tracking
- Weekly revision battle with leaderboard
- Performance analytics per subject/topic
