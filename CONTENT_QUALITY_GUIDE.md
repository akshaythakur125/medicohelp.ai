# Content Quality Guide

## MCQ Quality Standards

### Vignette Requirements
- **Length**: 6-7 lines of clinical information
- **Structure**: Age, gender, setting, presenting complaint, history, examination findings, investigation results
- **Key indicators**: "year-old", "presents", "history", "examination" must appear
- **Example**:
  > A 35-year-old female presents to the emergency department with a 3-day history of fever and right lower quadrant pain. On examination, there is tenderness at McBurney's point. Vitals: HR 100/min, BP 130/80, Temp 38.5°C. What is the most appropriate next step?

### Option Design
- 4 options (A, B, C, D) with one correct answer
- Distractors should be clinically plausible (close mimics, common confusions)
- Each distractor gets analysis explaining why it is wrong

### Answer Explanations
- Correct answer with reasoning
- Wrong-option analysis for each distractor
- Memory anchor/mnemonic appended
- Clinical pearl (50% chance)
- Cross-reference to related subject (30% chance)

## Format Quality Standards

### Rapid Revision
- Bullet-point format with 3-6 key facts
- High-yield takeaway included
- Clinical pearl or memory anchor appended

### Flashcards
- Front: question or concept name
- Back: answer with explanation (spoiler-tagged)

### True/False
- Statement derived from high-yield takeaway
- T/F verdict with explanation
- Clinical context included

### One-Liner Recall
- Fill-in-the-blank format from high-yield facts
- Blank positioned at a key term (typically middle of sentence)

### Mnemonics
- Acronym or phrase-based memory aid
- Clinical hook connecting to real exam scenarios

## Content Enrichment Pipeline
1. Base content from static library or AI generation
2. Subject-specific memory anchor appended
3. Clinical pearl (50% probability)
4. Cross-subject concept reference (30% probability)
5. MCQ distractor analysis (MCQ format only)
6. Bloom's taxonomy tag assignment
7. Engagement enrichment (hooks, exam tags)

## Anti-Duplication
- Recent title dedup: 100-entry LRU set
- Spaced repetition: 3-7 day cooldown per topic
- Strategy-level posted set: 500 titles
- Library content never repeats within cooldown window
